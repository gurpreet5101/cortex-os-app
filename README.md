# cortex-os-app
Cortex-os is a minimalist research journal designed for PhD scholars to organize ideas, log experiments, track failures, and build your research story in one place. Private by design, simple to use, and focused on thinking, not managing tools, now finally downloadable as an app!!!

# Download & Install
Go to the Releases page and download:
CORTEX OS Setup 3.0.0.exe: Installer — adds Start Menu & Desktop shortcut
CORTEX OS 3.0.0.exe:       Portable — no install needed, just double-click

Cortex OS is a research management system designed for PhD students and researchers. It brings together everything you need in one place:

-> Dashboard — today's tasks, hypotheses, and recent insights at a glance
-> Archive — store and annotate research papers with summaries, key ideas, and links to your work
-> Neural Log — an encrypted journal for thoughts, mood, and energy tracking with AI companion support
-> Hypothesis Tracker — manage ideas from conception through testing to conclusion
-> Failure Vault — log experiments that didn't work and extract lessons from them
-> Protocol Engine — define and document repeatable research procedures
-> Task Queue — prioritized task management with categories and statuses
-> Paper Tracker — track manuscripts from draft to publication
-> Advisor Sync — log meetings, feedback, and action items
-> Concept Graph — a visual brain map connecting your papers, ideas, and methods
-> Why — your mission statement and research timeline to keep you grounded
-> Reproducibility Vault — scripts, inputs, and step-by-step protocols so your work can be repeated

# How to Use

First Launch
Double-click the installer and follow the setup wizard
Open CORTEX OS from your Desktop or Start Menu
On first launch you'll see the Neural Link screen -> paste your Anthropic API key to enable the AI journal companion, or click Skip to use the app without AI features
Set your name, field, and institution in the Operative Config screen
Click Save Identity -> your profile is now set

Your API key is stored only on your local machine. Nothing is sent anywhere except to Anthropic when you use the AI features.

# Navigating the App

Use the sidebar on the left to switch between modules. The current active module is highlighted.
The Global Search bar at the top searches across all your data simultaneously.

# Adding Your First Paper (Archive)

Click Archive in the sidebar
Fill in the paper title, authors, journal, and year
Write a 3-line summary, the key idea, and how it affects your work — these are the most important fields
Add tags (comma separated) to make it searchable
Click Upload to Archive


# Logging a Hypothesis

Click Hypothesis in the sidebar
Write your hypothesis statement clearly
Set the status -> start with Idea and update it as you gather evidence
Add tags and link it to any existing experiments or protocols
Click Initialize Hypothesis

The tracker shows all hypotheses filterable by status: Promising, Testing, Ideas, Dead.

# Writing a Journal Entry (Neural Log)

Click Neural Log in the sidebar
Select your mood signature and energy level
Write your transmission — thoughts, progress, concerns, anything
Click Save Entry
Optionally click Consult Cortex AI to get a response from the AI companion (requires API key)

# Logging a Failure

Click Failure Vault in the sidebar
Fill in what the experiment was, what went wrong, and the root cause
The most important field: Lessons Learned — be specific
Click Vault This Failure

Every failure logged here is searchable and becomes future wisdom.

# Backing Up Your Data

All your data lives in local storage on your machine. To back it up:
Scroll to the bottom of any page to find Core
Click Export Backup — saves a .json file to your Downloads
To restore on a new machine, click Import Backup and select that file

Do this regularly. If you uninstall the app or clear browser data, local storage is wiped.

# System Tray

The app runs in the system tray (bottom-right of your taskbar).
Double-click the tray icon to bring the window back
Right-click the tray icon for options: Open, Quit

Closing the window minimizes it to the tray — it keeps running in the background.

# Privacy

Your Anthropic API key is stored in local storage, only on your machine
All research data (papers, hypotheses, journal entries, etc.) is stored locally
The AI features send your journal entries to the Anthropic API only when you click "Consult Cortex AI"
No data is collected, tracked, or sent anywhere else
